- Move the content to /c/Windows/Resources/Themes
- Use SecureUXTheme to patch this theme

To edit the theme:
Use msstyleEditor and navigate to DWMWindow.Common_Properties.IMAGEATLAS
You may have to edit the path variable in the .theme file
If you are having problems with title bar colors, you may have to change the accent color.
